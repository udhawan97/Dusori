# ADR-007: Vertical slice before AI

**Status:** accepted · **Date:** 2026-07-20

## Decision

Prove storage, portability, conflict safety, offline behavior, rendered notes, and the companion security boundary before implementing search, AI, model downloads, automated curricula, or scheduling.

## Consequences

The first public build has narrower features but a trustworthy product contract. Roadmap pages must label every excluded feature as planned rather than shipped.
