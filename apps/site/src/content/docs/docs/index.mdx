Dusori is a local-first learning workspace. The current build includes durable browser storage, an optional real-folder path on supported Chromium browsers, portable ZIP import/export, conflict-safe proposals, offline use, a local source library, preview-first curriculum import, consent-gated research through Microsoft Learn and English Wikipedia, an optional companion for confirmed readable-page extraction and ranked Microsoft Learn search, an interactive learning loop, and a portable knowledge graph derived from the same files.

The hosted app does **not** fetch arbitrary pages. The companion fetches only a URL source you explicitly confirm. Dusori does not extract PDFs, generate material with AI, use key-based or general web search, generate schedules, or perform unattended work.

[Read the v0.3.0 release notes](./releases/v0-3-0/) or [open the browser app](https://udhawan97.github.io/Dusori/app/).
