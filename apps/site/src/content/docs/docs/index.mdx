Dusori is a local-first learning workspace. The current build includes durable browser storage, an optional real-folder path on supported Chromium browsers, validated rollback-safe ZIP replacement, conflict-safe Markdown note authoring, local full-text search, backlinks and workspace health, offline use, a local source library, preview-first curriculum import, consent-gated research through Microsoft Learn and English Wikipedia, an optional companion for confirmed readable-page extraction and ranked Microsoft Learn search, a deterministic review queue and recap with an optional spaced-review schedule, and a portable knowledge graph derived from the same files.

The hosted app does **not** fetch arbitrary pages. The companion fetches only a URL source you explicitly confirm. Local workspace search never sends a query away. Dusori does not extract PDFs, generate material with AI, use key-based or general web search, or perform unattended work. Review scheduling is optional and exists only for topics you explicitly mark reviewed.
