Dusori is a local-first learning workspace. The current milestone includes durable browser storage, an optional real-folder path on supported Chromium browsers, portable ZIP import/export, conflict-safe proposals, offline use, a local source library, and preview-first curriculum import from pasted official or structured Markdown outlines.

It does **not** fetch syllabus pages, extract PDFs, generate material with AI, search the web, or schedule unattended work.
