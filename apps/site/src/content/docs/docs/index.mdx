Dusori v0.1.0 is a local-first learning workspace. The first public release includes durable browser storage, an optional real-folder path on supported Chromium browsers, portable ZIP import/export, conflict-safe proposals, offline use, a local source library, preview-first curriculum import, an interactive learning loop, and a portable knowledge graph derived from the same files.

It does **not** fetch syllabus pages, extract PDFs, generate material with AI, search the web, generate schedules, or perform unattended work.

[Read the v0.1.0 release notes](./releases/v0-1-0/) or [open the browser app](https://udhawan97.github.io/Dusori/app/).
