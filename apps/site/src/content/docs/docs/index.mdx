Dusori is a local-first learning workspace. The current milestone includes durable browser storage, an optional real-folder path on supported Chromium browsers, portable ZIP import/export, conflict-safe proposals, offline use, and a local source library for pasted text, Markdown/text files, and URL references.

It does **not** yet include AI generation, web search, automatic syllabus import, or unattended scheduling.
