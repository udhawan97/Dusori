Dusori is a local-first learning workspace. The current milestone includes durable browser storage, an optional real-folder path on supported Chromium browsers, portable ZIP import/export, conflict-safe proposals, offline use, a local source library, preview-first curriculum import, and an interactive learning loop derived from portable files.

It does **not** fetch syllabus pages, extract PDFs, generate material with AI, search the web, generate schedules, or perform unattended work.
